# WsylibBookRS
> **主要内容**:
解决目前学校图书管理系统不能由学生推荐图书到图书馆的问题,经过使用测试,符合生产使用环境.

> **使用技术**
- spring 4.3.10
- spring jdbc 4.3.10
- struts 2.5.3 
- mysql 5.7
> **开发环境**
- eclipse    maven 3.5
